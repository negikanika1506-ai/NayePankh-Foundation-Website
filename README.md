# NayePankh Foundation Website

## Overview

A responsive NGO website developed using HTML, CSS, and JavaScript. The website highlights the foundation's mission, programs, impact, volunteer opportunities, and contact information.

## Features

* Responsive Design
* Dark Mode
* Animated Gradient Hero Section
* Smooth Scrolling
* Hover Animations
* Impact Statistics Counter
* Gallery Section
* Contact Form
* Scroll To Top Button
* Mobile Friendly Layout

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Sections

* Home
* About Us
* Mission & Vision
* Programs
* Impact Statistics
* Success Stories
* Gallery
* Volunteer Section
* Contact
